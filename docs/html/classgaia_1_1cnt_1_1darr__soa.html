Detailed Description

template<typename T, typename Allocator = mem::DefaultAllocatorAdaptor>
class gaia::cnt::darr_soa< T, Allocator >

Array with variable size of elements of type.

Template Parameters
Tallocated on heap. Interface compatiblity with std::vector where it matters. Can be used only with SoA types.
