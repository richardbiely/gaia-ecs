Detailed Description

template<typename T, uint32_t PageCapacity, typename Allocator>
class gaia::cnt::sparse_storage< T, PageCapacity, Allocator, std::enable_if_t< std::is_empty_v< T > > >

Array with variable size of elements of type T allocated on heap. Allocates enough memory to support PageCapacity elements. Uses Allocator to allocate memory. This version is optimized for tags (data of zero size).

Template Parameters
TEmpty stored value type. param PageCapacity Number of sparse entries represented by each page. Must be a power of two.
AllocatorAllocator used by the storage pages.
